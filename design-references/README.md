# Design References

Esta pasta guarda as telas de referência usadas no redesign visual da Vita Ebooks.

Regras:

- O conteúdo aqui serve como referência de composição e atmosfera.
- Os HTMLs desta pasta não são o código-fonte produtivo do site público.
- Tailwind inline, imagens remotas e textos genéricos presentes nas referências não devem ser levados diretamente para produção.
- A implementação oficial do front público vive nas rotas da raiz do projeto, em `assets/css/style.css` e `assets/js/main.js`.
