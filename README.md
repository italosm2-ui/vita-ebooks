# Vita Ebooks

Plataforma de venda de ebooks digitais com foco em autodesenvolvimento, negócios e tecnologia. Desenvolvida como site estático com HTML, CSS e JavaScript puros.

## Estrutura do projeto

```
vita-ebooks/
├── index.html              ← Landing page completa
├── assets/
│   ├── css/
│   │   └── style.css       ← Design system + componentes
│   └── js/
│       └── main.js         ← Dark mode, menu mobile, dados mock
└── README.md
```

## Design

- **Tipografia:** Zodiak (display) + Satoshi (corpo) via Fontshare
- **Paleta:** Creme aquecido + verde-floresta `#1a5c3a` como único acento
- **Modo escuro:** Automático via `prefers-color-scheme` + toggle manual
- **Responsivo:** 375px → 1440px+
