# Fluxos — Vita Ebooks

## Fluxo 1: descoberta

`Home -> catálogo -> ebook`

## Fluxo 2: comparação comercial

`Home -> planos -> suporte`

## Fluxo 3: interesse em título específico

`Catálogo -> ebook -> suporte com slug pré-preenchido`

## Fluxo 4: leitura institucional

`Home -> sobre -> documentação / suporte`

## Fluxo 5: futuro da conta

`Home -> conta (em breve) -> suporte`

## Regra central

Enquanto não houver checkout e autenticação reais, todos os fluxos terminam em navegação informativa ou contato honesto.
